Fundamentos da linguagem HTML

Autor: Luiz C M de Oliveira

HTML significa - Hyper Text Markup Language , ou linguagem de marcação de hipertextos e é a linguagem utilizada para codificação de páginas web.

A linguagem HTML utiliza os chamados elementos que informam ao navegador como o conteúdo deve ser exibido.

Os elementos da linguagem HTML são formados por caracteres mnemônicos delimitados pelos símbolos < e >, denominados tags. Os elementos da lingugagem HTML podem ser compostos por somente tags de abertura, ou tags de abertura e fechamento, como veremos nos exemplos a seguir:

Tag - b - bold - Mostra o texto em negrito.

<b> texto </b> - texto

Tag - em emphasise - Mostra o texto em itálico.

<em> texto </em> - italico

Alguns elementos da linguagem HTML necessitam de parâmetros adicionais para poderem ser executados. Tais parâmetros são denominados atributos .

Exemplos de elementos que necessitam de atributos para serem interpretados:

<a href = "url"> link </a> - tag anchor que cria um texto definido pelo atributo href para direcionar o leitor e o chama de link .

<img src = "url" alt = "text"> link </a> - tag image que exibe a imagem hospedada no endereço "url" do atributo src . Caso a imagem não possa ser exibida, o valor "text" do atributo alt é exibida.

Os elementos da linguagem HTML podem ainda ser em bloco ou alinhados. Dos elementos estudados até o momento são alinhadas ( inline ) as tags: a, spam, br, img e strong e são em bloco as tags div e p.

Por fim a linguagem HTML apresenta inúmeros recursos e nem sempre o programador irá lembrar de todos. Para estes casos há inúmeros sites na internet que podem ser consultados. Uma ótima fonte de informações e exemplos está na página w3schools
